The {\ttfamily MP} class provides a model for simulating the thruster configuration and controls for a motion platform, specifically tailored for Autonomous Underwater Vehicles (AUVs). It comes with two types of thruster matrices, one for real-\/world operation and another for simulation. It also provides functionalities to update thruster outputs based on desired vehicle movements, map the thruster data to specific output ranges, and print out the current thruster data.
